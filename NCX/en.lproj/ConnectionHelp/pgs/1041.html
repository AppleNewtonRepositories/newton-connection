Importing information

When importing a file, Newton Connection examines the file extension to determine what type of data the file contains. For example, to import text the file must be named something.txt. Before you import, make sure the file you’re importing is in a format supported by Newton Connection. See this list for the file formats (translators) supported by Newton Connection.

To import a file:

Click a soup name in the Newton Connection window sidebar to view a soup.

Drag a file from the finder and drop it over the list of soup entries.

The information is imported to your Newton application.

You don’t have to be connected to a Newton device to import information to it. If you open a work file (by choosing File > Open Recent, for example) you can import information which will be sent to the Newton device when you sync next.
