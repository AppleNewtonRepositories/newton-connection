Installing packages

A package is a special ﬁle that adds features to your Newton device. Examples of packages include applications, electronic books, and system updates.

A package file should have the extension .newtonpkg in order to be recognised by Newton Connection. The extension .pkg is reserved by Mac OS X for its own Installer packages.

To install a package:

Choose File > Install Package…

Select a package file (or you can select more than one) and click Open.

The packages are installed on your Newton device. The status box in the Newton Connection window keeps you informed of progress.

You can also install one or more packages by dragging package icons to the Newton Connection application icon on your computer desktop or onto the Newton Connection window while the device or store view is shown.
