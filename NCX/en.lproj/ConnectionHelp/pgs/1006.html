Using different serial speeds

Newton Connection can use serial speeds higher than the 38,400 bps default provided by the Newton Dock application when you Connect via: Serial.

On your Newton device:

Install a package (for example, Ser115200.pkg) to provide that serial speed to the Dock app.

On your Macintosh computer:

Choose Newton Connection > Preferences.

Select the serial speed in the pop-up menu.

Close the Preferences window.

It is essential that both ends of the serial connection use the same speed. If you change the speed from 38400 you will need to select the right speed option in the Dock app when connecting.
