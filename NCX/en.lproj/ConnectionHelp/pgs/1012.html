Backing up information

A work file is automatically created for every Newton device you connect. This work file holds a copy of the information on the Newton device.

To back up the information on your Newton device:

Click the Sync button on the Newton Connection window.

The status box in the Newton Connection window keeps you informed of progress.

Information is synchronized: this means that changes made on the Newton device are added to the work file, and any changes made to the work file when not connected are sent to the Newton device.
