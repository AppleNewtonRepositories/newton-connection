About work files

Every time you sync, Newton Connection updates a work Ô¨Åle stored in your Application Support/Newton Connection folder. The work file contains the current state of the Newton device. There is a work file for each Newton device you connect to, named with the device’s serial number.

Backing up work files

If you want to save the state of your Newton device at a particular point, you can copy the work file into another folder for safe keeping. Then at a later date you can copy that work file back again and next time you sync or restore, your Newton device will be returned to that state.
