Setting a password

If desired, you can set a connection password for Newton Connection. This password prevents others from transferring information from your Newton device without your permission. The password you set works for all types of connection.

To set a connection password:

Choose Newton Connection > Preferences.

Click the Security icon.

Type your password.

Close the Preferences window.
