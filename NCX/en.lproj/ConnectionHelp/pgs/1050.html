Taking a screen shot

The Toolkit.pkg provided as part of the Newton Toolkit contains a function that captures an image of the Newton device’s screen and returns it to the desktop. Newton Connection uses this function to provide the same service.

Note: This means you must install the Toolkit.pkg on your Newton device before you can use the screenshot function. (The Toolkit.pkg is included with Newton Connection.)

To capture an image of your Newton device’s screen:

Click the Display item in the Newton Connection window sidebar.

Wait for the screen shot function to be established.

Arrange what’s displayed on your Newton device.

Click the Take Snapshot button in the Newton Connection window.

A screen image is captured and transferred to your Mac OS computer. Since the size of the image data is quite large, this can take some time.

Drag the image from the Newton Connection window into a document or onto the desktop.

You can also copy it onto the clipboard.
